# Quick Start Guide

Get TRIXTECH running in 5 minutes.

## Prerequisites

- Node.js 18+
- npm or yarn

## Setup & Run

### Terminal 1 - Backend

\`\`\`bash
cd backend
npm install
npm start
\`\`\`

Runs on: http://localhost:5000

### Terminal 2 - Frontend

\`\`\`bash
cd frontend
npm install
npm run dev
\`\`\`

Runs on: http://localhost:3000
